In dieser Liste findest du alle Funktionen, die dir auf dem Server zur Verfügung stehen. Um einen Befehl zu nutzen, tippe ihn mit einem Schrägstrich (z. B. **/help**) in den Chat

<Card title="Viele Befehle benötigen Argumente; diese werden hier in spitzen < > Klammern angegeben. Argumente in spitzen Klammern sind ein Platzhalter für z.B. Spielernamen, etc." icon="circle-info">
  Aus: /tell \<Spieler\> \<Nachricht\>\
  wird: /tell TheFrodoDE Was gab es heute bei dir leckeres zu Abend?\
  **Sind Argumente in eckigen \[ \] Klammern angegeben, so sind diese optional und müssen nicht angegeben werden.**
</Card>

## Nützliche Serverbefehle

| Befehl | Aliase | Beschreibung |
| --- | --- | --- |
| /sb |  | Schaltet das Scoreboard an bzw. aus |
| /money | /balance /bal | Zeigt dir deine aktuellen Münzen |
| /balancetop | /baltop | Zeigt dir die 10 wohlhabendsten Spieler |
| /pay \<Spieler\> \<Anzahl\> |  | Verschicke Münzen an einen Spieler |
| /jobs browse |  | Zeige dir die Aktiven Jobs an |

## Nützliche Chatbefehle

| Befehl | Aliase | Beschreibung |
| --- | --- | --- |
| /msg \<Spieler\> \<Nachricht\> | /tell | Flüstere einem Spieler |

## Teleportation

| Befehl | Aliase | Beschreibung |
| --- | --- | --- |
| /spawn |  | Teleportiert dich zum Spawn der Main-Welt |
| /farmwelt |  | Teleportiert dich zur normalen Farmwelt |
| /farmwelt back |  | Teleportiert dich zum letzten Punkt in der Main-Welt |
| /nether |  | Teleportiert dich in die Nether Farmwelt |
| /end |  | Teleportiert dich in die End Farmwelt |

## Claim Befehle

| Befehl | Aliase | Beschreibung |
| --- | --- | --- |
| /Trust \<Spieler\> | /t | Gewährt einem anderen Spieler Baurechte in deinem Claim. |
| /unTrust \<Spieler\> | /ut | Entfernt einen Spieler von deinem Grundstück (alle Rechte aufheben). |
| /AccessTrust \<Spieler\> | /at | Gewährt einem Spieler Zugriff auf Knöpfe, Hebel und Betten in deinem Claim. |
| /ContainerTrust \<Spieler\> | /ct | Gewährt einem Spieler Zugriff auf deine Knöpfe, Hebel, Betten, Werkzeuge, Container und Tiere. |
| /TrustList |  | Zeigt die Rechte für das aktuelle Gebiet an. |
| /PermissionTrust \<Spieler\> | /pt | Ermöglicht es einem Spieler, anderen den gleichen Zugriff zu gewähren, den er selbst hat. |
| /Untrust all |  | Löscht alle Rechte für alle Spieler in deinem Gebiet. |
| /UnTrust \<Spieler\> |  | Löscht alle Rechte für den Spieler in deinem Gebiet. |
| /AbandonClaim |  | Löscht den Claim, in dem du stehst. |
| /AbandonAllClaims |  | Löscht alle deine Claims. |
