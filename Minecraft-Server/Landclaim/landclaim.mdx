Zum erstellen einer Protection nimmt der Spieler die benötigte Goldschaufel in die Hand und wählt einen Bereich aus indem ihr zwei Punkte (Rechtsklick) mit der Goldschaufel Markiert. Durch das setzen einer Diagonale erstellt das Plugin daraus ein Rechteck.

<Accordion title="Das Tutorial mit Bildern">
  Du klickst mit der Goldenen Schaufel als erstes in eine Ecke deines Bereichs

  <Frame>
    ![Claim CreatePos1](/images/Claims/claim-createpos1.png)
  </Frame>

  Danach in die andere Ecke deines Bereichs, den du Claimen möchtest

  <Frame>
    <img
      alt="Claim CreatePos 2"
      lightAlt="Claim CreatePos 2"
      darkAlt="Claim CreatePos 2"
      src="/images/Claims/claim-createpos2.png"

  Damit hast du dein Claim angelegt. Nun kannst du mit /trust \<Spielername\> einem Spieler Baurechte auf deinem Claim geben. Alle weiteren Befehle findest du unter dem Reiter Befehle.

  Klickst du jetzt noch mit einem Stick auf dein Claim, siehst du wie groß dieses Claim ist.
</Accordion>

Bevor Ihr eine Protection anlegt solltet Ihr allerdings mit einem Stock überprüfen ob auf Eurem gewählten Bereich schon eine Protection vorhanden ist. Dazu klickt ihr einfach Rechtsklick mit einem Stick in der Hand auf den Boden, und es erscheint entweder die Nachricht

<Frame>
  ![ClaimInfo NoOwner](/images/Claims/claiminfo-noowner.png)
</Frame>

oder eine Nachricht, wenn ein Spieler dies schon geclaimt hat. Im Beispiel von einem Claim von TheFrodoDE

<Frame>
  ![ClaimInfo Owner](/images/Claims/claiminfo-owner.png)
</Frame>

Es wird auch angezeigt, wie groß dieses Claim ist.

<Frame>

## Claim Befehle

| Befehl | Aliase | Beschreibung |
| :-- | :-- | :-- |
| /Trust \<Spieler\> | /t | Gewährt einem anderen Spieler Baurechte in deinem Claim. |
| /unTrust \<Spieler\> | /ut | Entfernt einen Spieler von deinem Grundstück (alle Rechte aufheben). |
| /AccessTrust \<Spieler\> | /at | Gewährt einem Spieler Zugriff auf Knöpfe, Hebel und Betten in deinem Claim. |
| /ContainerTrust \<Spieler\> | /ct | Gewährt einem Spieler Zugriff auf deine Knöpfe, Hebel, Betten, Werkzeuge, Container und Tiere. |
| /TrustList |  | Zeigt die Rechte für das aktuelle Gebiet an. |
| /PermissionTrust \<Spieler\> | /pt | Ermöglicht es einem Spieler, anderen den gleichen Zugriff zu gewähren, den er selbst hat. |
| /Untrust all |  | Löscht alle Rechte für alle Spieler in deinem Gebiet. |
| /UnTrust \<Spieler\> |  | Löscht alle Rechte für den Spieler in deinem Gebiet. |
| /AbandonClaim |  | Löscht den Claim, in dem du stehst. |
| /AbandonAllClaims |  | Löscht alle deine Claims. |
